# 合并排序好的两个数组
给定两个已经排序好的数组nums1(m个)和nums2(n个),现将nums2合并到nums1中，并且合并后的数组也是有序的。

方法1：将nums2中的元素逐步插入到nums1中。在num1中找到插入位置，将后面的元素移动一个位置。
